# media-portal-badge

A captive media portal badge stack.

# Requirements

* [Git](https://git-scm.com/)
* [Ruby](https://www.ruby-lang.org/en/)
* [Bundler](https://bundler.io/)
* [ISC DHCP Server](https://www.isc.org/downloads/dhcp/)
